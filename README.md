
# 📊 Telecom Customer Churn Analysis (EDA)

## 🔍 Project Overview
This project performs an **Exploratory Data Analysis (EDA)** on a Telecom Customer Churn dataset to understand
customer behavior and identify key factors that influence churn.  
The insights from this analysis can help telecom companies design better retention strategies.

## 📁 Dataset Description
The dataset contains customer-level information such as:
- Demographics (Gender, Senior Citizen, Partner, Dependents)
- Services subscribed (Internet, Phone, Streaming, Security, etc.)
- Contract and billing details
- Monthly and total charges
- Target variable: **Churn**

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## 📈 Key Analysis Performed
- Data cleaning and preprocessing
- Univariate and bivariate analysis
- Churn distribution analysis
- Impact of contract type, payment method, and internet service on churn
- Visualization using count plots, pie charts, and bar plots

## 🔑 Key Insights
- Customers with **month-to-month contracts** show significantly higher churn.
- **Electronic check** payment users have the highest churn rate.
- Customers without **online security and tech support** are more likely to churn.
- Long-tenure customers tend to stay with the service.

## 📌 Conclusion
Contract type, payment method, tenure, and add-on services play a crucial role in customer churn.
These insights can support predictive modeling and business decision-making.
